# Museum of Life

A simple landing page design inspired by an art-museum/gallery layout, built with plain HTML/CSS/JS.

## Structure

- `index.html` — page markup
- `styles.css` — styling
- `script.js` — smooth-scroll for nav links

## Notes

- "Museum of Life" hero title uses the **Alex Brush** script font (Google Fonts), matching the flowing script style from the design inspiration.
- The name "Nuha Hammad" in the top-right of the header uses **Times New Roman**.
- Hero and collection images are placeholders (via placehold.co) — swap them out with real artwork/photos in `index.html` once chosen.

## Viewing locally

Just open `index.html` in a browser, or serve the folder with any static file server.
